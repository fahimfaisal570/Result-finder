# Result Finder PRO 🏛️🚀

A state-of-the-art academic analytics engine and dashboard. Sync your web experience with absolute CLI-native accuracy and dynamic multi-batch discovery.

## ✨ Features
- **Manual Multi-Range Discovery**: Add multiple registration ranges and sessions in a single scan.
- **CLI-Native Accurate Scanning**: Lenient detection (GPA/Subjects) ensures 100% data extraction.
- **Split-Table Reporting**: Categorizes results into "Main Batch" and "Senior Re-adds" automatically.
- **Auto-Open Interaction**: Dropdowns float open on hover for a seamless web experience.
- **Android Support**: Fully compatible with Pydroid 3 (run `cli_scraper.py` directly).

## 🛠️ Setup & Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    cd YOUR_REPO_NAME
    ```
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Launch the Dashboard**:
    ```bash
    streamlit run app.py
    ```

## 🧹 Project Structure
- `app.py`: The Main Web Dashboard.
- `cli_scraper.py`: The Core Scraping Engine & CLI interface.
- `requirements.txt`: Python dependencies.
- `.gitignore`: Repository safety filters.
